# AGENTS.md

## Scope

This repository is dedicated to the digital-brain / multidimensional emotion memory project.

## Working Principles

- Treat this repository as project-specific, not as a generic ops workspace.
- Preserve architecture, task ledgers, and continuity files as source-of-truth for multi-agent collaboration.
- Prefer adding project-specific files under dedicated directories instead of mixing unrelated host-ops materials.
- Do not store secrets, passwords, tokens, or private keys in this repository.

## Collaboration

- Update `memory/digital-brain-project-board.md` when task status changes.
- Update `memory/session-handoff.md` before pausing after substantial progress.
- Keep reusable architecture and design documents under `knowledge/digital-brain/`.
